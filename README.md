# Chronic Kidney Disease Data Analysis – CRISP-DM Project

## Project Overview
This project applies the CRISP-DM methodology to analyze healthcare data and support predictive modeling for chronic kidney disease risk assessment. The goal is to extract meaningful insights from clinical datasets and evaluate patterns associated with disease presence.

## Dataset
- Structured healthcare dataset
- Clinical and laboratory variables
- Binary classification target (disease / no disease)
- ~70K+ processed records

## Methodology
- Data Understanding & Exploratory Data Analysis
- Data Cleaning & Preprocessing
- Feature Exploration
- Statistical Analysis
- Predictive Modeling
- Model Evaluation

## Tools & Technologies
Python | Pandas | NumPy | Scikit-learn | Matplotlib | Seaborn | CRISP-DM

## Project Structure
data/ – dataset
notebooks/ – analysis and modeling scripts
